Common questions & Useful resources about game development

So, What is game development?

It is a work area which is all about creating games! It doesn't have to include coding inside of it. There are several areas that allows someone work on game development without doing any coding.

Example areas:

  • 3D character modeller
  • Game Designer
  • Sound engineer
  • Game Scene Writer

    • What are the negative sides of game development areas?

    It has a small job pool and generally, people do it because they want to work on that topic so they compete alot and they don't care about money that much. So it earns lesser compared to other development areas.
